Pertemuan 1: Pengenalan Web

Pengenalan Web

Pertemuan ini membahas konsep dasar World Wide Web (WWW), proses kerja web, serta pengenalan HTML sebagai bahasa markup untuk membangun halaman web.

  • World Wide Web (WWW) adalah kumpulan halaman web di seluruh dunia yang saling terhubung melalui hyperlink dan dapat diakses melalui internet menggunakan browser.
  • Web bekerja dengan konsep client–server, di mana browser sebagai client meminta data dan server mengirimkan halaman web.
  • Komunikasi antara client dan server menggunakan protokol HTTP.
  • Browser berfungsi menerjemahkan dokumen HTML menjadi tampilan yang dapat dibaca pengguna.
  • HTML (Hypertext Markup Language) digunakan untuk menyusun struktur halaman web.
  • Struktur dasar HTML terdiri dari tag <html>, <head>, dan <body>.

Pertemuan 2: Basic Tag HTML

Struktur dan Tag Dasar HTML

Pertemuan ini membahas struktur penulisan dokumen HTML serta penggunaan tag dasar yang digunakan untuk membangun halaman web.

  • Dokumen HTML memiliki struktur utama yang terdiri dari tag <html>, <head>, dan <body>.
  • Bagian <head> berisi informasi pendukung seperti judul halaman (<title>), meta tag, CSS, dan JavaScript.
  • Bagian <body> berisi seluruh konten yang ditampilkan pada halaman web.
  • Tag heading (<h1> sampai <h6>) digunakan untuk judul dengan tingkatan berbeda.
  • Tag paragraf (<p>) digunakan untuk menampilkan teks.
  • Tag <br> digunakan untuk membuat baris baru, sedangkan <hr> untuk membuat garis pemisah.
  • List terurut (<ol>) dan tidak terurut (<ul>) digunakan untuk menampilkan daftar.

Pertemuan 3: Format Teks dan Karakter Khusus

Format Teks dan Karakter Khusus

Pertemuan ini membahas penggunaan tag HTML untuk mengatur format teks serta penulisan karakter khusus agar tampilan halaman web lebih informatif dan mudah dibaca.

  • Format teks dasar meliputi <b> dan <strong> untuk teks tebal, <i> dan <em> untuk teks miring, serta <u> untuk garis bawah.
  • Tag <small> dan <big> digunakan untuk mengatur ukuran teks relatif terhadap ukuran normal.
  • Tag <sub> (subscript) dan <sup> (superscript) digunakan untuk penulisan rumus atau notasi khusus.
  • Tag <s> atau strike digunakan untuk mencoret teks.
  • Marquee digunakan untuk membuat teks berjalan pada halaman web.
  • Karakter khusus HTML ditulis menggunakan entity, seperti &copy; (©), &reg; (®), &lt; (<), dan &gt; (>).

Pertemuan 4: Hyperlink

Tujuan Pembelajaran:
Mahasiswa mampu memahami dan mempraktikkan penggunaan hyperlink pada halaman web.

Pengertian Hyperlink

Hyperlink adalah fasilitas pada HTML yang digunakan untuk menghubungkan teks atau gambar ke halaman lain, bagian tertentu pada halaman yang sama, website lain, maupun ke email.

Jenis-Jenis Hyperlink

  1. Link Relatif
    Digunakan untuk berpindah halaman dalam satu website.
    <a href="profil.html">Profil</a>
  2. Link Absolut
    Digunakan untuk menuju website lain.
    <a href="https://www.google.com">Google</a>
  3. Link Anchor
    Digunakan untuk menuju bagian tertentu pada halaman yang sama.
    <a href="#materi">Ke Materi</a>
  4. Link Email
    Digunakan untuk membuka aplikasi email.
    <a href="mailto:email@domain.com">Kirim Email</a>

Kesimpulan

Hyperlink sangat penting dalam pembuatan website karena memudahkan navigasi dan menghubungkan informasi antar halaman.
